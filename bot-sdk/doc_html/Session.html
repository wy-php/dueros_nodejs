Class: Session

Session()

new Session()

封装对session的操作。DuerOS提过了多轮对话的能力，也能替Bot管理多轮对话，无须Bot自己维护session状态。 同时，也提供了Session存储，Bot还可以将对话的状态保存session中，自己处理多轮逻辑。 但是，存储在session中的数据，DuerOS是无法将其自动应用到下一轮的query解析中，对话状态的维护需要Bot自己完成
